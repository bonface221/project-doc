Setup Development Environment

In order to kickstart the development, the first is to setup a react development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site http://nodejs.org/

Apis

  • Where to provide data to the frontend
    1. supply data to the charts in the (/src/variables)
    2. supply global data to the components using redux or create context (/src/variables)
    3. consume the data in the components using useContext or using redux

Introduction

To install triform ui dashboard on your local machine, download the zip file provided and unzip them to the local disk on your computer. Open the folder with a code editor like vscode. And follow the stapes below-

we will use npm:

Open vscode terminal
Then run : npm install
Then run : npm start
Now in the browser go localhost:3000

Files structure

  • public - All static files
    • images :favicon image, manifest.json
    • index.html - Static index page.
  • src - Content all the react file
    • assets - All images and svgs
    • components - All Components regarding the project
    • Layout - Layout for the Admin route, and auth route
    • Theme - theme styles and configs
    • Variables Barchat and Linechart variables
    • Views Views for the App
    • index.js - Entry posing of react app
    • routes.js - All the routes for the app
  • package.json - All packages have information
  • node_modules - After installation you can seee this folder, this folder content all installed package.
